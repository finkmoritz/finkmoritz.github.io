C# Battle Card Game Framework

Implement custom battle card game mechanics.
Easy to use and fast to develop with.

About

The C# Battle Card Game Framework is the perfect match for developers that would love to create their own battle card game (such as Hearthstone or Magic The Gathering) but hate to do all the hard work until an implementation can actually be played.

This framework already provides the basic setup that is needed to make such a game come to live. So the developer can concentrate on the actual mechanics of the cards and their effects.

The framework is being developed in C# language and thus can be easily integrated into Unity projects.

Get Started

You can add the C# Battle Card Game Framework as NuGet package or directly clone it from GitHub.
Follow the tutorial to implement your first game!
